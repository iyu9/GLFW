## 実行方法

### Windows:

1. GLFW.vsproj以下のプロジェクトファイルを開く
2. glfw, glewのdll, lib参照に注意して実行

### Mac:
- GLFW.xcode以下のプロジェクトファイルを開く
- またはmake実行, ./RunApp

### Linux:
- make -f makefile_linux
